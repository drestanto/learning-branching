Repositori ini adalah tempat latihan interaktif untuk menguasai **Git Branching**, manajemen *workflow*, dan taktik penyelesaian *Merge Conflict*. Di sini, setiap *branch* adalah dimensi paralel yang siap dijelajahi!

## 🗺️ QUEST MAP (MISSION PROGRESS)

Ikuti peta misi di bawah ini untuk menamatkan *game* pembelajaran ini:

- [x] **🎚️ Level 1:** Menginisialisasi Repositori & `main` branch.
- [ ] **🌿 Level 2:** Membuat cabang waktu baru (`feature/new-skill`).
- [ ] **⚔️ Level 3:** Melakukan *Merge* kembali ke `main` tanpa merusak berkas.
- [ ] **🔥 Boss Fight:** Sengaja membuat dan menyelesaikan *Merge Conflict*!

---

## 🛠️ SPELL BOOK (ESSENTIAL COMMANDS)

Gunakan mantra-mantra CLI berikut untuk memanipulasi ruang dan waktu di dalam repositori ini:

### 🌟 Fase 1: Membuka Gerbang Dimensi (Create & Switch)
*   `git branch` — *Melihat dimensi (branch) yang aktif saat ini.*
*   `git checkout -b <nama-branch>` — *Membuka gerbang paralel baru dan langsung melompat ke dalamnya.*

### 🔗 Fase 2: Menyatukan Garis Waktu (Merge)
*   `git switch main` — *Kembali ke dimensi utama.*
*   `git merge <nama-branch>` — *Menarik seluruh pencapaian dari dimensi lain ke dimensi utama.*

### 🧹 Fase 3: Pembersihan Kosmik (Delete)
*   `git branch -d <nama-branch>` — *Menghapus cabang waktu yang sudah tidak digunakan lagi agar repositori tetap bersih.*
